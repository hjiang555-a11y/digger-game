# Digger Game (挖金子)

HTML5 Canvas arcade game inspired by Windmill Software's Digger (1983). Control the big-mouth character, dig through dirt, collect gold bags, avoid purple Nobbin enemies. 8 levels, Web Audio chiptune sound effects.

## How to Play

Open `index.html` in a browser. Arrow keys to move.

## Tech

Pure HTML5 Canvas + JavaScript, no dependencies.
